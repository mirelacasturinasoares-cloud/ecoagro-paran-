# ecoagro-paran-
Plataforma web interativa desenvolvida para o Concurso Agrinho 2026 (Subcategoria: Programação Front-End). O projeto aborda o tema "Agro forte, futuro sustentável", destacando o equilíbrio entre produção e meio ambiente no Paraná por meio de tecnologias nativas (HTML5, CSS3 e JavaScript), incluindo simulador, quiz educativo e gráficos dinâmicos.
